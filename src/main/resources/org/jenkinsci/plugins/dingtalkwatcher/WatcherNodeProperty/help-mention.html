atUserIds 被@人的用户userid。多个之间用英文逗号分割. 提醒群中的指定成员(@某个成员)，如果开发者获取不到userid，可以使用手机号列表.

atMobiles 手机号列表，多个之间用英文逗号分割. 提醒手机号对应的群成员(@某个成员). 注意 只有在群内的成员才可被@，非群内成员手机号会被脱敏。

@all 表示提醒所有人, 

    "at": {
      "atMobiles": [
          "150XXXXXXXX"
      ],
      "atUserIds": [
          "user123"
      ],
      "isAtAll": false
    }
